01 产品概述

1.1 产品定位

MirrorKit（镜箱） 是一款面向中国开发者的 macOS 原生应用，用于统一管理和切换开发工具镜像源。它的核心价值是将分散在终端命令、环境变量、git 配置中的镜像配置工作，整合为一个直观的图形界面。

产品 slogan：极速切换开发工具镜像源 — 以镜为鉴，可明得失。

1.2 解决的问题

中国开发者在日常工作中频繁遇到以下问题：

  • 访问慢/超时：GitHub clone 失败、brew update 卡住、pip install 超时、npm install 缓慢
  • 配置碎片化：每个工具配置镜像的方式不同（环境变量、git remote、pip.conf、.npmrc 等），记忆成本高
  • 镜像发现难：各镜像站（清华 TUNA、中科大 USTC、华为云、腾讯云 等）的可用性和延迟不一，手动对比困难
  • 恢复原状麻烦：调试完毕后忘记改回官方源，或不知如何彻底清除镜像配置

1.3 平台与技术栈

维度选择
平台macOS 26.4+（2026 年）
UI 框架SwiftUI + Observation 框架
架构模式MVVM（Model-View-ViewModel）
设计语言Prism Design System（自定义）
持久化UserDefaults + Shell Profile 文件写入

02 目标用户与场景

2.1 用户画像

👨‍💻 前端/全栈开发者

日常使用 npm/yarn、GitHub、Node.js。频繁遇到 npm install 超时和 git clone 失败。

🐍 Python 开发者

依赖 pip install 安装包，受 PyPI 官方源速度影响大。熟悉国内镜像站但厌倦手动配置。

🍺 macOS 重度用户

通过 Homebrew 管理开发工具，brew update/install 经常因 GitHub 访问不畅而失败。

🦀 Rust/Flutter 开发者

工具链安装和依赖下载依赖国外 CDN，国内网络环境严重影响开发效率。

🎓 高校学生

使用教育网，部分镜像站可能不可用。需要测速功能找到当前可用的最快源。

3.2 镜像切换机制

工具切换方式配置文件生效时机
Homebrew 环境变量 HOMEBREW_API_DOMAIN, HOMEBREW_BOTTLE_DOMAIN + git remote set-url ~/.zshrc / ~/.bash_profile 新终端
GitHub git config --global url.insteadOf ~/.gitconfig 全局即时
Oh My Zsh git remote set-url origin ~/.oh-my-zsh/.git/config 即时
PyPI pip config set global.index-url ~/.config/pip/pip.conf 即时
Node.js 环境变量 NPM_CONFIG_REGISTRY / NODE_OPTIONS ~/.zshrc / ~/.bash_profile 新终端
Rustup 环境变量 RUSTUP_DIST_SERVER, RUSTUP_UPDATE_ROOT ~/.zshrc / ~/.bash_profile 新终端

3.3 测速机制

  • 方式：对每个镜像源的关键 URL 发起 HEAD 请求，测量响应时间
  • 频率：进入页面自动测速一次，用户可手动刷新（⌘R）
  • 展示：延迟以毫秒显示，配合可视化延迟条（相对比例）
  • 缓存：测速结果在页面生命周期内保留，切换页面或 app 重启后重新测速

5.3 异常流程

  • 切换失败：显示错误通知横幅，操作日志记录失败详情，不修改 activeMirrorId
  • 测速失败：该镜像显示 "N/A"，不影响其他镜像的切换操作
  • Flutter 未安装/非 git repo：显示引导页或跳过 SDK remote 操作
  • 取消确认弹窗：不执行任何操作，pendingMirror 置 nil

06 设计系统 — Prism

6.1 设计原则

  • 沉浸暗色：以深色为基底，营造专注、精致的工具感
  • 信息优先：减少视觉干扰，内容层级清晰，核心操作一目了然
  • 一致性：每个工具页面使用同一套布局模板和组件
  • 原生质感：遵循 Apple 设计规范，不使用自定义控件取代原生行为

6.2 色彩体系

08 交互规范

8.1 确认弹窗策略

策略适用工具原因
需要确认Brew, GitHub涉及 git 全局配置或环境变量修改，影响范围广
直接切换Oh My Zsh, PyPI, Node.js, Rustup操作可逆，影响范围限定

8.2 帮助弹窗

  • 每个工具页面标题右侧显示 questionmark.circle 图标按钮
  • 点击弹出 popover，包含：功能说明 + 使用要点（bullet list，约 3 条）
  • 宽度 360px，背景色 prismSurface
  • 点击外部关闭

8.3 键盘快捷键

快捷键功能
⌘R当前页面重新测速

8.4 通知横幅

  • 切换成功后显示顶部通知横幅，3 秒自动消失
  • 成功：success 样式（绿色）
  • 失败：error 样式（红色）
  • 点击可主动关闭
