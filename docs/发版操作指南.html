本地发版操作指南

适用于 macOS App + GitHub Releases

前置条件

  • Xcode 26+
  • macOS Sequoia 26.4+
  • GitHub CLI: brew install gh
  • create-dmg: brew install create-dmg

发版步骤

1. 更新版本号

在 Xcode 项目设置中更新：

  • Version (CFBundleShortVersionString): 1.0.0
  • Build (CFBundleVersion): 递增 1

路径：Target → General → Version & Build

2. 创建 Git Tag

git tag v1.0.0
git push origin v1.0.0

3. 构建 Archive

xcodebuild archive \
  -scheme mirrorkit \
  -configuration Release \
  -archivePath ./build/mirrorkit.xcarchive

4. 导出为 .app

创建 export.plist：

<?xml version="1.0" encoding="UTF-8"?>
<!DOCTYPE plist PUBLIC "-//Apple//DTD PLIST 1.0//EN"
  "http://www.apple.com/DTDs/PropertyList-1.0.dtd">
<plist version="1.0">
<dict>
    <key>method</key>
    <string>developer-id</string>
    <key>signingStyle</key>
    <string>automatic</string>
</dict>
</plist>
xcodebuild -exportArchive \
  -archivePath ./build/mirrorkit.xcarchive \
  -exportPath ./build/export \
  -exportOptionsPlist export.plist

5. 创建 DMG

create-dmg \
  --volname "MirrorKit" \
  --window-pos 200 120 \
  --window-size 600 400 \
  --icon-size 100 \
  --icon "MirrorKit.app" 175 190 \
  --hide-extension "MirrorKit.app" \
  --app-drop-link 425 190 \
  "MirrorKit-v1.0.0.dmg" \
  "./build/export/MirrorKit.app"

6. 上传到 GitHub Releases

gh release create v1.0.0 \
  ./MirrorKit-v1.0.0.dmg \
  --title "MirrorKit v1.0.0" \
  --notes "版本更新内容..."

一键发版脚本

将以下内容保存为 scripts/release.sh：

#!/bin/bash
set -e

VERSION="$1"
if [ -z "$VERSION" ]; then
  echo "Usage: ./scripts/release.sh v1.0.0"
  exit 1
fi

echo "=== Building $VERSION ==="
xcodebuild archive \
  -scheme mirrorkit \
  -configuration Release \
  -archivePath ./build/mirrorkit.xcarchive

xcodebuild -exportArchive \
  -archivePath ./build/mirrorkit.xcarchive \
  -exportPath ./build/export \
  -exportOptionsPlist export.plist

create-dmg \
  --volname "MirrorKit" \
  --window-pos 200 120 \
  --window-size 600 400 \
  --icon-size 100 \
  --icon "MirrorKit.app" 175 190 \
  --hide-extension "MirrorKit.app" \
  --app-drop-link 425 190 \
  "MirrorKit-$VERSION.dmg" \
  "./build/export/MirrorKit.app"

gh release create "$VERSION" \
  "./MirrorKit-$VERSION.dmg" \
  --title "MirrorKit $VERSION" \
  --notes ""
